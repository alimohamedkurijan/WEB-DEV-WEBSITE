üß≠ What is Flexbox?

Flexbox is a modern CSS layout method that makes it easy to create flexible, responsive layouts without using floats or positioning.

Why Flexbox?

  • Problem it solves: Difficult layouts with floats or inline-block
  • Flex container: Parent element with display: flex
  • Flex items: Direct children that become flexible
  • Real-life analogy: Organizing boxes inside a shelf
Basic Flexbox Structure 
/* HTML */
<div class="flex-container">
    <div class="flex-item">Item 1</div>
    <div class="flex-item">Item 2</div>
    <div class="flex-item">Item 3</div>
</div>

/* CSS */
.flex-container {
    display: flex;
}

üõ†Ô∏è display: flex

The display: flex property transforms a container into a flex container, making its direct children flex items.

What happens with display: flex:

  • Direct children become flex items
  • Items flow in a row by default
  • Items can grow and shrink to fit container
  • No more float or inline-block issues
Display Flex Example 
/* Before: Items stack vertically */
.container {
    /* No display property */
}

/* After: Items align horizontally */
.container {
    display: flex;
    background-color: #f0f0f0;
    padding: 20px;
}

.flex-item {
    background-color: #3498db;
    color: white;
    padding: 15px;
    margin: 5px;
}

üîÅ flex-direction

Controls the direction in which flex items are laid out within the flex container.

flex-direction Values:

  • row: Items flow left to right (default)
  • row-reverse: Items flow right to left
  • column: Items flow top to bottom
  • column-reverse: Items flow bottom to top
Flex Direction Examples 
/* Horizontal layout (default) */
.container {
    display: flex;
    flex-direction: row;
}

/* Vertical layout */
.container {
    display: flex;
    flex-direction: column;
}

/* Reverse horizontal */
.container {
    display: flex;
    flex-direction: row-reverse;
}

/* Reverse vertical */
.container {
    display: flex;
    flex-direction: column-reverse;
}

üìè justify-content

Controls how flex items are distributed along the main axis (horizontal alignment in row direction).

justify-content Values:

  • flex-start: Items at the beginning (default)
  • center: Items centered
  • flex-end: Items at the end
  • space-between: Space between items
  • space-around: Space around items
  • space-evenly: Equal space between and around items
Justify Content Examples 
/* Center items horizontally */
.navbar {
    display: flex;
    justify-content: center;
}

/* Space items evenly */
.card-container {
    display: flex;
    justify-content: space-between;
}

/* Center with equal spacing */
.button-group {
    display: flex;
    justify-content: space-evenly;
}

üìê align-items

Controls how flex items are aligned along the cross axis (vertical alignment in row direction).

align-items Values:

  • stretch: Items stretch to fill container (default)
  • flex-start: Items at the top
  • center: Items centered vertically
  • flex-end: Items at the bottom
  • baseline: Items aligned by their text baseline
Align Items Examples 
/* Center items vertically */
.card-row {
    display: flex;
    align-items: center;
    height: 200px;
}

/* Align items to bottom */
.footer {
    display: flex;
    align-items: flex-end;
    height: 100px;
}

/* Stretch items to full height */
.sidebar {
    display: flex;
    flex-direction: column;
    align-items: stretch;
}

üß± Combine Layout Concepts

Now let's combine all these properties to create practical layouts.

Common Layout Patterns:

  • Navigation bar: Horizontal layout with centered items
  • Card layout: Responsive grid with equal spacing
  • Sidebar layout: Vertical navigation with full height
  • 3-section layout: Header, content, footer structure
Complete Layout Example 
/* Navigation Bar */
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem;
    background-color: #333;
    color: white;
}

/* Card Container */
.card-container {
    display: flex;
    justify-content: space-around;
    align-items: stretch;
    flex-wrap: wrap;
    gap: 20px;
    padding: 20px;
}

/* 3-Section Layout */
.page-layout {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}

.header {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 80px;
    background-color: #2c3e50;
}

.content {
    display: flex;
    flex: 1;
}

.sidebar {
    display: flex;
    flex-direction: column;
    width: 250px;
    background-color: #ecf0f1;
}

.main-content {
    flex: 1;
    padding: 20px;
}

.footer {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 60px;
    background-color: #34495e;
}
