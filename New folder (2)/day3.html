🎯 What is CSS?

CSS = Cascading Style Sheets - the language that controls the visual presentation of your HTML elements.

HTML vs CSS Analogy:

  • HTML: The building structure (walls, rooms, foundation)
  • CSS: The paint, decor, layout, and styling
Basic CSS Structure 
/* CSS Rule Structure */
selector {
    property: value;
    another-property: value;
}

/* Example */
h1 {
    color: blue;
    font-size: 24px;
}

🧩 Inline vs Internal vs External CSS

There are three ways to apply CSS to your HTML elements, each with its own use case.

Three CSS Methods:

  • Inline: style="" inside HTML tags
  • Internal: <style> inside <head>
  • External: .css file linked with <link>
CSS Methods Examples 
/* 1. Inline CSS */
<p style="color: red; font-size: 16px;">Red text</p>

/* 2. Internal CSS */
<head>
    <style>
        p {
            color: blue;
            font-size: 18px;
        }
    </style>
</head>

/* 3. External CSS */
<head>
    <link rel="stylesheet" href="styles.css">
</head>

🎯 Selectors, Properties, Values

CSS selectors target HTML elements, and properties define how they should be styled.

Selector Types:

  • Tag selectors: p {}, h1 {}, div {}
  • Class selectors: .myClass {} (reusable)
  • ID selectors: #myId {} (unique)
Selector Examples 
/* Tag selector - targets all paragraphs */
p {
    color: #333;
    line-height: 1.6;
}

/* Class selector - targets elements with class="highlight" */
.highlight {
    background-color: yellow;
    font-weight: bold;
}

/* ID selector - targets element with id="header" */
#header {
    background-color: #2c3e50;
    color: white;
    padding: 20px;
}

🖌️ Styling Essentials

Master these fundamental CSS properties to create beautiful, well-styled web pages.

Essential Properties:

  • Colors: color, background-color
  • Typography: font-family, font-size, line-height
  • Borders: border, border-radius
  • Spacing: margin (outside), padding (inside)
Styling Examples 
/* Colors */
.text-primary {
    color: #3498db;
    background-color: #ecf0f1;
}

/* Typography */
.heading {
    font-family: 'Arial', sans-serif;
    font-size: 24px;
    line-height: 1.4;
}

/* Borders */
.card {
    border: 2px solid #bdc3c7;
    border-radius: 8px;
}

/* Spacing */
.container {
    margin: 20px;
    padding: 15px;
}

🆚 Class vs ID

Understanding when to use classes vs IDs is crucial for writing maintainable CSS.

Key Differences:

  • Classes (.class): Reusable, multiple elements can have same class
  • IDs (#id): Unique, only one element per page can have that ID
  • Specificity: IDs have higher specificity than classes
Class vs ID Usage 
/* HTML */
<div id="header">Website Header</div>
<div class="card">Card 1</div>
<div class="card">Card 2</div>
<div class="card">Card 3</div>

/* CSS */
#header {
    background-color: #2c3e50;
    color: white;
    padding: 20px;
}

.card {
    background-color: #fff;
    border: 1px solid #ddd;
    padding: 15px;
    margin: 10px;
    border-radius: 5px;
}
