üíª What is HTML?

HTML (HyperText Markup Language) is the foundation of all web pages. Think of it as the skeleton of a website - just like bones give structure to your body, HTML gives structure to web pages.

üìö Key Points:

  • Structure: HTML defines the structure and content of web pages
  • Markup: Uses tags to "mark up" content with meaning
  • Universal: Every website you visit uses HTML
  • Browser Rendering: Browsers read HTML and display it as formatted pages
Raw HTML vs Browser Display 
<h1>My First Website</h1>
<p>This is a paragraph with <strong>bold text</strong>.</p>
<a href="https://example.com">Click here</a>

Browser Display:

My First Website

This is a paragraph with bold text.

Click here

üß± Basic HTML Structure

Every HTML document follows a standard structure with essential tags that tell the browser how to interpret the content.

üèóÔ∏è Essential Tags:

  • <!DOCTYPE html> - Declares this is an HTML5 document
  • <html> - The root element containing everything
  • <head> - Contains metadata (title, styles, scripts)
  • <body> - Contains the visible page content
Complete HTML Structure 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is the main content of my page.</p>
</body>
</html>

üí° Pro Tip:

Proper indentation makes your HTML code more readable. Each nested element should be indented with 2 or 4 spaces.

‚úçÔ∏è Text Tags

HTML provides various tags to structure and format text content, making it readable and meaningful.

üìù Text Elements:

  • Headings: <h1> to <h6> for titles and subtitles
  • Paragraphs: <p> for text blocks
  • Bold: <strong> (semantic) or <b> (visual)
  • Italic: <em> (semantic) or <i> (visual)
Text Formatting Example 
<h1>Main Title</h1>
<h2>Subtitle</h2>
<p>This is a <strong>paragraph</strong> with <em>emphasized text</em>.</p>
<p>You can also use <b>bold</b> and <i>italic</i> for visual styling.</p>

üéØ Best Practice:

Use <strong> and <em> for semantic meaning (importance/emphasis), and <b> and <i> only for visual styling.

üîó Links

Links (hyperlinks) connect web pages together and are fundamental to the web experience.

üîó Link Types:

  • Absolute URLs: Complete web addresses (https://example.com)
  • Relative URLs: Paths relative to current page (./page.html)
  • Internal Links: Links to sections on the same page (#section)
Link Examples 
<!-- External link -->
<a href="https://www.google.com">Visit Google</a>

<!-- Link opening in new tab -->
<a href="https://www.google.com" target="_blank">Open in New Tab</a>

<!-- Internal link -->
<a href="#section1">Jump to Section 1</a>

<!-- Email link -->
<a href="mailto:contact@example.com">Send Email</a>

üéØ Practice Exercise:

Create a navigation menu with links to different sections of your page:

<nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
    <a href="#contact">Contact</a>
</nav>

üìã Lists

Lists help organize information in a structured way, making content easier to read and understand.

üìã List Types:

  • Ordered Lists: <ol> for numbered items
  • Unordered Lists: <ul> for bullet points
  • List Items: <li> for individual items
List Examples 
<!-- Ordered List (numbered) -->
<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>

<!-- Unordered List (bullets) -->
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Orange</li>
</ul>

<!-- Nested Lists -->
<ul>
    <li>Fruits
        <ul>
            <li>Apple</li>
            <li>Banana</li>
        </ul>
    </li>
    <li>Vegetables
        <ul>
            <li>Carrot</li>
            <li>Broccoli</li>
        </ul>
    </li>
</ul>

üí° Real-World Uses:

  • Recipe ingredients and steps
  • Navigation menus
  • Feature lists
  • Checklists and to-dos

üñºÔ∏è Images

Images make websites visually appealing and help convey information more effectively.

üñºÔ∏è Image Attributes:

  • src - Source URL of the image
  • alt - Alternative text for accessibility
  • width and height - Image dimensions
  • title - Tooltip text on hover
Image Examples 
<!-- Basic image -->
<img src="photo.jpg" alt="A beautiful sunset">

<!-- Image with dimensions -->
<img src="photo.jpg" alt="A beautiful sunset" width="300" height="200">

<!-- Image with title -->
<img src="photo.jpg" alt="A beautiful sunset" title="Sunset at the beach">

<!-- External image -->
<img src="https://example.com/image.jpg" alt="External image">

‚ôø Accessibility Importance:

The alt attribute is crucial for:

  • Screen readers for visually impaired users
  • Displaying when images fail to load
  • SEO (Search Engine Optimization)
  • Better user experience
