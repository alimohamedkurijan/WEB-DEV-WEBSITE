üíª What is JavaScript?

JavaScript is a powerful programming language that brings websites to life. Think of it as the brain of a website - just like your brain controls your body, JavaScript controls the behavior and interactivity of web pages.

üìö Key Points:

  • Dynamic: JavaScript makes websites interactive and responsive
  • Versatile: Can run in browsers, servers, and mobile apps
  • Essential: Every modern website uses JavaScript
  • Real-time: Updates content without refreshing the page
Simple JavaScript Example 
// This is a JavaScript comment
let message = "Hello, World!";
console.log(message);

// Interactive example
let name = prompt("What's your name?");
alert("Welcome, " + name + "!");

What happens:

1. Prints "Hello, World!" to console

2. Asks for your name in a popup

3. Shows a welcome message

Variables & Data Types

Variables are containers that store data. JavaScript has different types of data, each with specific characteristics and uses.

üîß Variable Declaration:

  • let - Can be changed (reassigned)
  • const - Cannot be changed (constant)
  • var - Old way (avoid using)
Variables and Data Types 
// String - text data
let name = "Ali";
const city = "Kuwait";

// Number - numeric data
let age = 25;
const pi = 3.14159;

// Boolean - true/false values
let isStudent = true;
const isWorking = false;

// Output methods
console.log("Name:", name);
alert("Welcome to " + city);
let userInput = prompt("Enter your age:");

üí° Pro Tip:

Use const by default, and only use let when you need to change the value later. This makes your code more predictable and prevents accidental changes.

üßÆ Mathematical Operations

JavaScript can perform all basic mathematical operations and more. You can use it as a calculator and combine numbers with other data types.

üî¢ Basic Operators:

  • Addition: + (also concatenates strings)
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Modulus: % (remainder)
Math Operations Examples 
// Basic calculations
let a = 10;
let b = 5;

let sum = a + b;        // 15
let difference = a - b; // 5
let product = a * b;    // 50
let quotient = a / b;   // 2
let remainder = a % b;  // 0

// String concatenation
let firstName = "Ali";
let lastName = "Ahmed";
let fullName = firstName + " " + lastName; // "Ali Ahmed"

// Combining strings and numbers
let age = 25;
let message = "I am " + age + " years old"; // "I am 25 years old"

console.log("Sum:", sum);
console.log("Full name:", fullName);
console.log("Message:", message);

üéØ Building Interactive Apps

Now let's combine everything we've learned to create a simple but functional application that interacts with users.

üõ†Ô∏è App Building Steps:

  • Input: Collect data from users
  • Process: Manipulate and calculate with the data
  • Output: Display results to users
  • Enhance: Add more features and validation
Mini Greeting App 
function createGreeting() {
    // Step 1: Get user input
    let name = prompt("What's your name?");
    let age = prompt("How old are you?");
    
    // Step 2: Process the data
    age = parseInt(age); // Convert string to number
    let nextYearAge = age + 1;
    
    // Step 3: Create personalized message
    let greeting = "Hello " + name + "!";
    let ageMessage = "You are " + age + " years old.";
    let futureMessage = "Next year you'll be " + nextYearAge + "!";
    
    // Step 4: Display results
    alert(greeting + "\n" + ageMessage + "\n" + futureMessage);
    
    // Step 5: Log for debugging
    console.log("User:", name, "Age:", age);
}

// Run the app
createGreeting();

üí° Pro Tip:

Always use parseInt() or Number() when converting user input to numbers, as prompt() always returns a string, even for numbers.
