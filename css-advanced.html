🌈 CSS Gradients

CSS Gradients allow you to create smooth transitions between two or more colors, creating beautiful visual effects.

Gradients can be used for backgrounds, borders, and even text effects, adding depth and visual interest to your designs.

🔴 Linear Gradients

  • linear-gradient(to right, red, blue)
  • linear-gradient(45deg, #ff6b6b, #4ecdc4)
  • Direction-based color transitions

⭕ Radial Gradients

  • radial-gradient(circle, red, blue)
  • radial-gradient(ellipse, #ff6b6b, #4ecdc4)
  • Center-outward color transitions
/* Linear gradient examples */
.gradient-bg {
  background: linear-gradient(45deg, #667eea, #764ba2);
}

/* Radial gradient examples */
.circle-gradient {
  background: radial-gradient(circle, #f093fb, #f5576c);
}

/* Multiple color stops */
.rainbow-gradient {
  background: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);
}

🔄 CSS Transforms

CSS transforms allow you to modify the appearance and position of elements without affecting the document flow.

🎯 2D Transforms

  • translate(): Move elements horizontally and vertically
  • rotate(): Rotate elements around their center
  • scale(): Make elements larger or smaller
  • skew(): Distort elements by tilting them

📐 3D Transforms

  • rotateX() Rotate around X-axis
  • rotateY() Rotate around Y-axis
  • rotateZ() Rotate around Z-axis
  • translateZ() Move along Z-axis

🔧 Transform Origin

  • center Default center point
  • top left Top-left corner
  • bottom right Bottom-right corner
  • 50% 50% Custom percentage
/* 2D transform examples */
.rotate-element {
  transform: rotate(45deg);
}

.scale-element {
  transform: scale(1.5);
}

/* Multiple transforms */
.complex-transform {
  transform: translate(50px, 100px) rotate(45deg) scale(1.2);
}

/* 3D transforms */
.rotate-3d {
  transform: rotateX(45deg) rotateY(30deg);
  transform-style: preserve-3d;
}

🎬 CSS Animations

CSS animations bring your websites to life with smooth, performant animations that enhance user experience.

⏱️ Transitions

  • transition: all 0.3s ease
  • transition-property: transform
  • transition-duration: 0.5s
  • transition-timing-function: ease-in-out

🎭 Keyframes

  • @keyframes slideIn Define animation sequence
  • 0% Starting state
  • 50% Middle state
  • 100% Ending state

⚙️ Animation Properties

  • animation-name: Name of the keyframe animation
  • animation-duration: How long the animation takes
  • animation-timing-function: Speed curve of the animation
  • animation-delay: Delay before animation starts
  • animation-iteration-count: How many times to repeat
  • animation-direction: Forward, backward, or alternating
/* Transition example */
.button {
  transition: all 0.3s ease;
}

.button:hover {
  transform: scale(1.1);
  background-color: #ff6b6b;
}

/* Keyframe animation */
@keyframes bounce {
  0%, 20%, 50%, 80%, 100% {
    transform: translateY(0);
  }
  40% {
    transform: translateY(-30px);
  }
  60% {
    transform: translateY(-15px);
  }
}

.bouncing-element {
  animation: bounce 2s infinite;
}

📱 Responsive Design

Responsive design ensures your websites look and work perfectly on all devices, from mobile phones to desktop computers.

📐 Media Queries

  • Mobile-first: Design for mobile first, then enhance for larger screens
  • Breakpoints: Common breakpoints: 768px (tablet), 1024px (desktop)
  • Flexible layouts: Use relative units (%, em, rem) instead of fixed pixels
  • Flexible images: Images that scale with their containers

📱 Mobile-First Approach

  • Start with mobile layout
  • Use min-width media queries
  • Progressive enhancement
  • Touch-friendly interactions

🖥️ Desktop Enhancements

  • Larger navigation menus
  • Multi-column layouts
  • Hover effects
  • More detailed content
/* Mobile-first responsive design */
.container {
  width: 100%;
  padding: 0 15px;
  margin: 0 auto;
}

/* Tablet breakpoint */
@media (min-width: 768px) {
  .container {
    max-width: 750px;
  }
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Desktop breakpoint */
@media (min-width: 1024px) {
  .container {
    max-width: 970px;
  }
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

/* Large desktop */
@media (min-width: 1200px) {
  .container {
    max-width: 1170px;
  }
}
