1

üéØ What is Flexbox?

Flexbox (Flexible Box Layout) is a CSS layout model designed for creating responsive and flexible web layouts.

It provides an efficient way to distribute space among items in a container and align them, even when their size is unknown or dynamic.

üí° Key Benefits

  • Responsive: Automatically adjusts to different screen sizes
  • Flexible: Items can grow, shrink, and reorder as needed
  • Alignment: Easy centering and distribution of elements
  • Direction: Can flow in rows or columns

Flexbox is perfect for navigation bars, card layouts, forms, and any component that needs to be responsive and flexible.

2

üèóÔ∏è Flexbox Container

To use flexbox, you need to create a flex container by setting display: flex on an element.

üìã Container Properties

  • display: flex - Creates a flex container
  • flex-direction - Sets the main axis direction
  • justify-content - Aligns items along main axis
  • align-items - Aligns items along cross axis
.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
  gap: 1rem;
}

The flex container becomes the parent element that controls how its direct children (flex items) are laid out.

3

üìê Flex Direction & Alignment

Flexbox provides powerful alignment and direction control for your layouts:

üîÑ Flex Direction

  • row - Items flow left to right (default)
  • row-reverse - Items flow right to left
  • column - Items flow top to bottom
  • column-reverse - Items flow bottom to top

üéØ Justify Content

  • flex-start - Items at start (default)
  • center - Items centered
  • flex-end - Items at end
  • space-between - Equal spacing

üìè Align Items

  • stretch - Items stretch to fill (default)
  • center - Items centered
  • flex-start - Items at start
  • flex-end - Items at end

üîÑ Wrap

  • nowrap - No wrapping (default)
  • wrap - Wrap to next line
  • wrap-reverse - Wrap in reverse
4

üì¶ Flex Items

Individual flex items can have their own properties to control how they behave within the container:

üîß Item Properties

  • flex-grow: How much an item can grow relative to others
  • flex-shrink: How much an item can shrink relative to others
  • flex-basis: The initial size of an item
  • align-self: Override container's align-items for specific item
  • order: Change the visual order of items

üìù Common Values

.flex-item {
  flex: 1; /* shorthand for flex-grow: 1 */
  flex: 0 1 auto; /* flex-grow: 0, flex-shrink: 1, flex-basis: auto */
  flex: 2 1 200px; /* grow: 2, shrink: 1, basis: 200px */
}

üí° Use Cases

  • Equal width columns
  • Responsive navigation
  • Card layouts
  • Form layouts

The flex shorthand property combines flex-grow, flex-shrink, and flex-basis in one declaration.
