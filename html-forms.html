Introduction to Forms

Forms are the backbone of user interaction on the web. They allow users to submit data, make selections, and interact with your website.

Form Structure:

  • : Container element that wraps all form controls
  • action: Specifies where the form data should be sent
  • method: Defines how the data should be sent (GET or POST)

Radio & Checkbox Inputs

Radio buttons and checkboxes allow users to make selections from predefined options.

Key Differences:

  • Radio: Only one option can be selected (use same name attribute)
  • Checkbox: Multiple options can be selected
