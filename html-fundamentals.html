üíª What is HTML?

HTML (HyperText Markup Language) is the foundation of all web pages. Think of it as the skeleton of a website - just like bones give structure to your body, HTML gives structure to web pages.

üìö Key Points:

  • ‚Ä¢ Structure: HTML defines the structure and content of web pages
  • ‚Ä¢ Markup: Uses tags to "mark up" content with meaning

  • ‚Ä¢ Browser Rendering: Browsers read HTML and display it as formatted pages

üîó Links

Links (hyperlinks) connect web pages together and are fundamental to the web experience.

üîó Link Types:

  • ‚Ä¢ Absolute URLs: Complete web addresses (https://example.com)
  • ‚Ä¢ Relative URLs: Paths relative to current page (./page.html)
  • ‚Ä¢ Internal Links: Links to sections on the same page (#section)
Link Examples 
<!-- External link -->
<a href="https://www.google.com">Visit Google</a>

<!-- Link opening in new tab -->
<a href="https://www.google.com" target="_blank">Open in New Tab</a>

<!-- Internal link -->
<a href="#section1">Jump to Section 1</a>

<!-- Email link -->
<a href="mailto:contact@example.com">Send Email</a>

Images

Images make websites visually appealing and help convey information more effectively.

Image Attributes:

  • ‚Ä¢ src - Source URL of the image
  • ‚Ä¢ alt - Alternative text for accessibility
  • ‚Ä¢ width and height - Image dimensions
  • ‚Ä¢ title - Tooltip text on hover
Image Examples 
<!-- Basic image -->
<img src="photo.jpg" alt="A beautiful sunset">

<!-- Image with dimensions -->
<img src="photo.jpg" alt="A beautiful sunset" width="300" height="200">

<!-- Image with title -->
<img src="photo.jpg" alt="A beautiful sunset" title="Sunset at the beach">

<!-- External image -->
<img src="https://example.com/image.jpg" alt="External image">
