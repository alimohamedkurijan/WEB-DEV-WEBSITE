üíª What is HTML?

HTML (HyperText Markup Language) is the foundation of all web pages. Think of it as the skeleton of a website - just like bones give structure to your body, HTML gives structure to web pages.

üìö Key Points:

  • Structure: HTML defines the structure and content of web pages
  • Markup: Uses tags to "mark up" content with meaning
  • Universal: Every website you visit uses HTML
  • Browser Rendering: Browsers read HTML and display it as formatted pages

üîó Links

Links (hyperlinks) connect web pages together and are fundamental to the web experience.

üîó Link Types:

  • Absolute URLs: Complete web addresses (https://example.com)
  • Relative URLs: Paths relative to current page (./page.html)
  • Internal Links: Links to sections on the same page (#section)

Images

Images make websites visually appealing and help convey information more effectively.

Image Attributes:

  • src - Source URL of the image
  • alt - Alternative text for accessibility
  • width and height - Image dimensions
  • title - Tooltip text on hover
