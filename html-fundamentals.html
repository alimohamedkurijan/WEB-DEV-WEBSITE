🎯 What is HTML?

HTML = HyperText Markup Language

HTML is the standard markup language for creating web pages. It describes the structure of web content using a system of elements and tags.

💡 Key Concepts

  • Markup: HTML uses tags to mark up content
  • Structure: Defines the layout and organization of web pages
  • Semantic: Provides meaning to content for browsers and screen readers

HTML is the foundation of every website - it's what gives structure to your content and makes it readable by web browsers.

2

🏗️ Basic HTML Structure

Every HTML document follows a standard structure with essential elements:

📋 Essential Elements

  • <!DOCTYPE html> - Document type declaration
  • <html> - Root element
  • <head> - Contains metadata
  • <body> - Contains visible content
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>
3

🧩 HTML Elements & Tags

HTML elements are the building blocks of web pages. Each element has an opening tag, content, and closing tag.

📝 Text Elements

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <strong> - Bold text
  • <em> - Italic text

🔗 Link Elements

  • <a> - Hyperlinks
  • <img> - Images
  • <video> - Video content
  • <audio> - Audio content

📋 List Elements

  • <ul> - Unordered lists
  • <ol> - Ordered lists
  • <li> - List items

🏗️ Structure Elements

  • <div> - Content division
  • <section> - Section content
  • <header> - Header content
  • <footer> - Footer content
4

🎯 Semantic HTML

Semantic HTML uses meaningful tags that clearly describe their purpose to both browsers and developers.

💡 Why Use Semantic HTML?

  • Accessibility: Screen readers can better understand your content
  • SEO: Search engines can better index your pages
  • Maintainability: Code is easier to read and maintain
  • Future-proof: Better compatibility with new technologies

✅ Semantic Elements

  • <nav> - Navigation
  • <main> - Main content
  • <article> - Article content
  • <aside> - Sidebar content

❌ Non-semantic Elements

  • <div> - Generic container
  • <span> - Generic inline
  • <b> - Bold (use <strong> instead)
  • <i> - Italic (use <em> instead)
