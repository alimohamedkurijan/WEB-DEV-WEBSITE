💻 What is HTML?

HTML (HyperText Markup Language) is the foundation of all web pages. Think of it as the skeleton of a website - just like bones give structure to your body, HTML gives structure to web pages.

📚 Key Points:

  • • Structure: HTML defines the structure and content of web pages
  • • Markup: Uses tags to "mark up" content with meaning
  • • Universal: Every website you visit uses HTML
  • • Browser Rendering: Browsers read HTML and display it as formatted pages
Raw HTML vs Browser Display 
<h1>My First Website</h1>
<p>This is a paragraph with <strong>bold text</strong>.</p>
<a href="https://example.com">Click here</a>

This is a paragraph with bold text.

🔗 Links

Links (hyperlinks) connect web pages together and are fundamental to the web experience.

🔗 Link Types:

  • • Absolute URLs: Complete web addresses (https://example.com)
  • • Relative URLs: Paths relative to current page (./page.html)
  • • Internal Links: Links to sections on the same page (#section)
Link Examples 
<!-- External link -->
<a href="https://www.google.com">Visit Google</a>

<!-- Link opening in new tab -->
<a href="https://www.google.com" target="_blank">Open in New Tab</a>

<!-- Internal link -->
<a href="#section1">Jump to Section 1</a>

<!-- Email link -->
<a href="mailto:contact@example.com">Send Email</a>

Practice Exercise:

Create a navigation menu with links to different sections of your page:

<nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
    <a href="#contact">Contact</a>
</nav>

Images

Images make websites visually appealing and help convey information more effectively.

Image Attributes:

  • • src - Source URL of the image
  • • alt - Alternative text for accessibility
  • • width and height - Image dimensions
  • • title - Tooltip text on hover
Image Examples 
<!-- Basic image -->
<img src="photo.jpg" alt="A beautiful sunset">

<!-- Image with dimensions -->
<img src="photo.jpg" alt="A beautiful sunset" width="300" height="200">

<!-- Image with title -->
<img src="photo.jpg" alt="A beautiful sunset" title="Sunset at the beach">

<!-- External image -->
<img src="https://example.com/image.jpg" alt="External image">

♿ Accessibility Importance:

The alt attribute is crucial for:

  • • Screen readers for visually impaired users
  • • Displaying when images fail to load
  • • SEO (Search Engine Optimization)
  • • Better user experience
