üì¶ What are Arrays?

Arrays are ordered collections that can store multiple values in a single variable. Think of them as shopping lists - you can have one list with many items, and each item has its own position!

üìö Key Points:

  • Ordered: Items have specific positions (indexes)
  • Flexible: Can store different types of data
  • Dynamic: Can add or remove items easily
  • Indexed: Access items using position numbers (starting from 0)
Array Creation and Access 
// Creating arrays
let fruits = ["apple", "banana", "orange"];
let numbers = [1, 2, 3, 4, 5];
let mixed = ["text", 42, true, null];

// Accessing elements (indexing starts at 0)
console.log(fruits[0]);        // "apple"
console.log(fruits[1]);        // "banana"
console.log(fruits[2]);        // "orange"

// Getting array length
console.log(fruits.length);    // 3

// Accessing last element
console.log(fruits[fruits.length - 1]); // "orange"

What happens:

1. Creates arrays with different data types

2. Accesses elements by index (0-based)

3. Gets array length with .length property

4. Accesses last element dynamically

üîß Modifying Arrays

Arrays are dynamic - you can add and remove items easily. The most common methods are .push() to add items and .pop() to remove them.

üõ†Ô∏è Common Array Methods:

  • push(): Add item to the end of array
  • pop(): Remove and return last item
  • unshift(): Add item to the beginning
  • shift(): Remove and return first item
  • splice(): Add/remove items at any position
Array Modification Examples 
let todoList = ["Learn JavaScript", "Build a project"];

// Adding items
todoList.push("Practice arrays");
console.log(todoList); // ["Learn JavaScript", "Build a project", "Practice arrays"]

todoList.unshift("Wake up early");
console.log(todoList); // ["Wake up early", "Learn JavaScript", "Build a project", "Practice arrays"]

// Removing items
let completedTask = todoList.pop();
console.log("Completed:", completedTask); // "Practice arrays"
console.log(todoList); // ["Wake up early", "Learn JavaScript", "Build a project"]

let firstTask = todoList.shift();
console.log("First task:", firstTask); // "Wake up early"
console.log(todoList); // ["Learn JavaScript", "Build a project"]

// Modifying specific positions
todoList.splice(1, 0, "Take a break"); // Insert at index 1
console.log(todoList); // ["Learn JavaScript", "Take a break", "Build a project"]

üí° Pro Tip:

Use .push() and .pop() for stack-like behavior (last in, first out). Use .unshift() and .shift() for queue-like behavior (first in, first out).

üîÑ Understanding Loops

Loops allow you to repeat code multiple times. When working with arrays, loops let you process every item without writing the same code over and over.

üîÑ Loop Types:

  • for loop: Traditional loop with counter
  • forEach: Modern array method for iteration
  • for...of: ES6 loop for iterating values
  • while: Loop while condition is true
Loop Examples 
let fruits = ["apple", "banana", "orange", "grape"];

// Traditional for loop
console.log("For loop:");
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

// forEach method (modern and cleaner)
console.log("forEach method:");
fruits.forEach(function(fruit) {
    console.log(fruit);
});

// forEach with arrow function (ES6)
console.log("forEach with arrow function:");
fruits.forEach(fruit => console.log(fruit));

// for...of loop (ES6)
console.log("for...of loop:");
for (let fruit of fruits) {
    console.log(fruit);
}

// Processing with loops
let numbers = [1, 2, 3, 4, 5];
let sum = 0;

for (let i = 0; i < numbers.length; i++) {
    sum += numbers[i];
}
console.log("Sum:", sum); // 15

üéØ When to Use Each Loop

Different loops are better for different situations. Understanding when to use each type makes your code more readable and efficient.

üéØ Loop Selection Guide:

  • forEach: When you need to do something with each item
  • for loop: When you need the index or more control
  • for...of: When you only need the values (cleaner than forEach)
  • while: When you don't know how many iterations you need
Loop Use Cases 
let students = ["Ali", "Sarah", "Ahmed", "Fatima"];

// forEach: Simple iteration
students.forEach(student => {
    console.log("Hello, " + student + "!");
});

// for loop: Need index
for (let i = 0; i < students.length; i++) {
    console.log("Student " + (i + 1) + ": " + students[i]);
}

// for...of: Clean value iteration
for (let student of students) {
    console.log("Welcome, " + student);
}

// Processing with index
let scores = [85, 92, 78, 96];
let total = 0;

for (let i = 0; i < scores.length; i++) {
    total += scores[i];
    console.log("Score " + (i + 1) + ": " + scores[i]);
}

let average = total / scores.length;
console.log("Average score:", average);

üõí Building a Shopping Cart

Now let's combine everything we've learned to create a practical shopping cart application that demonstrates arrays and loops in action.

üõ†Ô∏è Cart Features:

  • Add Items: Use push() to add products
  • Remove Items: Use pop() or splice()
  • Display Cart: Use loops to show all items
  • Calculate Total: Loop through items to sum prices
  • Format Output: Create user-friendly messages
Complete Shopping Cart 
// Shopping cart simulation
let cart = [];
let products = [
    { name: "Laptop", price: 999 },
    { name: "Mouse", price: 25 },
    { name: "Keyboard", price: 75 },
    { name: "Monitor", price: 299 }
];

// Function to add item to cart
function addToCart(productIndex) {
    if (productIndex >= 0 && productIndex < products.length) {
        cart.push(products[productIndex]);
        console.log("Added " + products[productIndex].name + " to cart");
    }
}

// Function to remove last item
function removeFromCart() {
    if (cart.length > 0) {
        let removed = cart.pop();
        console.log("Removed " + removed.name + " from cart");
    } else {
        console.log("Cart is empty!");
    }
}

// Function to display cart
function displayCart() {
    if (cart.length === 0) {
        console.log("Your cart is empty!");
        return;
    }
    
    console.log("=== SHOPPING CART ===");
    let total = 0;
    
    // Loop through cart items
    for (let i = 0; i < cart.length; i++) {
        let item = cart[i];
        console.log((i + 1) + ". " + item.name + " - $" + item.price);
        total += item.price;
    }
    
    console.log("Total items: " + cart.length);
    console.log("Total price: $" + total);
    console.log("====================");
}

// Test the shopping cart
addToCart(0); // Add laptop
addToCart(1); // Add mouse
addToCart(2); // Add keyboard
displayCart();
removeFromCart(); // Remove keyboard
displayCart();

üí° Pro Tip:

When working with arrays of objects, use forEach or for...of for cleaner code. Use traditional for loops when you need the index or more complex control flow.
