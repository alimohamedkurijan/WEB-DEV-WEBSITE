üéØ What is JavaScript?

JavaScript is a programming language that adds interactivity and dynamic behavior to web pages.

It's the language that makes websites come alive - from simple form validation to complex web applications and games.

üí° Key Features

  • Client-side: Runs in the user's browser
  • Dynamic: Can change content and behavior on the fly
  • Event-driven: Responds to user actions and events
  • Versatile: Used for frontend, backend, and mobile apps

JavaScript is essential for modern web development and works alongside HTML (structure) and CSS (styling) to create complete web experiences.

2

üì¶ Variables & Data Types

Variables are containers that store data values. JavaScript has several data types:

üî¢ Numbers

  • let age = 25;
  • let price = 19.99;
  • Includes integers and decimals

üìù Strings

  • let name = "John";
  • let message = 'Hello World';
  • Text enclosed in quotes

‚úÖ Booleans

  • let isActive = true;
  • let isLoggedIn = false;
  • true or false values

üìã Arrays

  • let colors = ["red", "green", "blue"];
  • let numbers = [1, 2, 3, 4, 5];
  • Ordered collections of values
// Variable declaration examples
let firstName = "John";
const age = 30;
var isStudent = true;

// Data type checking
console.log(typeof firstName); // "string"
console.log(typeof age); // "number"
console.log(typeof isStudent); // "boolean"

üî¢ Operators & Math

JavaScript provides various operators for performing calculations and comparisons:

üßÆ Arithmetic Operators

  • + Addition: 5 + 3 = 8
  • - Subtraction: 10 - 4 = 6
  • * Multiplication: 6 * 7 = 42
  • / Division: 15 / 3 = 5
  • % Modulus: 17 % 5 = 2

üîç Comparison Operators

  • == Equal to (loose)
  • === Equal to (strict)
  • != Not equal to
  • > Greater than
  • < Less than

üîó Logical Operators

  • && AND (both must be true)
  • || OR (either can be true)
  • ! NOT (inverts boolean)
// Math operations
let x = 10;
let y = 5;

let sum = x + y; // 15
let difference = x - y; // 5
let product = x * y; // 50
let quotient = x / y; // 2
let remainder = x % y; // 0

// Comparison
let isGreater = x > y; // true
let isEqual = x === y; // false
4

üîÑ Control Flow

Control flow statements allow you to make decisions and repeat actions in your code:

üéØ If Statements

if (age >= 18) {
  console.log("Adult");
} else if (age >= 13) {
  console.log("Teenager");
} else {
  console.log("Child");
}

üîÑ Loops

// For loop
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// While loop
let count = 0;
while (count < 3) {
  console.log(count);
  count++;
}

üí° Common Patterns

  • Conditional Logic: Use if/else to make decisions
  • Iteration: Use loops to repeat actions
  • Switch Statements: Alternative to multiple if/else
  • Ternary Operator: Shorthand for simple conditions
