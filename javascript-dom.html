üå≥ What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page as a tree of objects that JavaScript can manipulate.

Think of the DOM as a family tree where each HTML element is a node, and JavaScript can access, modify, add, or remove these nodes to create dynamic web experiences.

üí° DOM Tree Structure

Document (Root)
HTML
Head (Title, Meta, CSS)
Body (Content)
Div, P, Button (Elements)
// DOM represents this HTML structure:
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <div id="main">
      <p>Hello World</p>
    </div>
  </body>
</html>

üîç DOM Selection Methods

JavaScript provides several methods to select elements from the DOM, each with different use cases and capabilities.

üÜî ID Selection

  • getElementById('id')
  • Returns single element
  • Fastest method
  • Most specific

üè∑Ô∏è Class Selection

  • getElementsByClassName('class')
  • Returns HTMLCollection
  • Multiple elements
  • Good for styling groups

üè∑Ô∏è Tag Selection

  • getElementsByTagName('tag')
  • Returns HTMLCollection
  • All elements of type
  • Useful for bulk operations

üéØ Modern Selection

  • querySelector('selector')
  • CSS-style selectors
  • Returns first match
  • Very flexible
// Different selection methods
const elementById = document.getElementById('myId');
const elementsByClass = document.getElementsByClassName('myClass');
const elementsByTag = document.getElementsByTagName('p');
const elementByQuery = document.querySelector('.myClass');
const allElementsByQuery = document.querySelectorAll('.myClass');

// CSS-style selectors with querySelector
const firstButton = document.querySelector('button');
const specificButton = document.querySelector('button.primary');
const nestedElement = document.querySelector('#container .item');

üéØ Event Handling

Events are actions that happen in the browser, such as clicks, form submissions, or page loads. JavaScript can listen for these events and respond accordingly.

üñ±Ô∏è Common Event Types

  • Mouse Events: click, dblclick, mouseover, mouseout, mousedown, mouseup
  • Keyboard Events: keydown, keyup, keypress
  • Form Events: submit, change, input, focus, blur
  • Document Events: DOMContentLoaded, load, resize, scroll

üìù Event Listeners

  • addEventListener()
  • Modern approach
  • Multiple handlers
  • Better control

üîó Inline Events

  • onclick="function()"
  • Simple setup
  • Single handler
  • Less flexible
// Adding event listeners
const button = document.querySelector('#myButton');

// Method 1: addEventListener (recommended)
button.addEventListener('click', function() {
  console.log('Button clicked!');
});

// Method 2: Inline event
// <button onclick="handleClick()">Click me</button>

// Method 3: Property assignment
button.onclick = function() {
  console.log('Button clicked!');
};

// Event object provides information about the event
button.addEventListener('click', function(event) {
  console.log('Event:', event);
  console.log('Target:', event.target);
  console.log('Type:', event.type);
});

üîß DOM Manipulation

DOM manipulation allows you to dynamically change the content, structure, and styling of web pages without reloading the page.

üìù Content Changes

  • innerHTML HTML content
  • textContent Text only
  • innerText Visible text

üé® Style Changes

  • style.property Inline styles
  • classList CSS classes
  • setAttribute() Attributes

üèóÔ∏è Creating Elements

  • createElement(): Create new HTML elements
  • appendChild(): Add elements to the DOM
  • removeChild(): Remove elements from the DOM
  • cloneNode(): Copy existing elements
// Changing content
const element = document.querySelector('#myElement');
element.innerHTML = '<strong>New content</strong>';
element.textContent = 'Plain text content';

// Changing styles
element.style.backgroundColor = 'red';
element.style.fontSize = '20px';
element.classList.add('highlight');
element.classList.remove('old-class');

// Creating new elements
const newDiv = document.createElement('div');
newDiv.textContent = 'New element';
newDiv.className = 'new-class';

// Adding to DOM
document.body.appendChild(newDiv);

// Removing elements
const parent = element.parentNode;
parent.removeChild(element);
